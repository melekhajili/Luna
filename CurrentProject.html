Website Project


The website project is a teamwork project that needs to be created and developed with using HTML, CSS, and JavaScript. Before starting building the site, my team and I set a deadline for ourselves to complete the individual part, so that later we could build a site from scratch with the knowledge we gained. The individual part was that each of us had to register on the Codeacademy website and complete 3 online courses: HTML, CSS and Java Script. After completing all the courses, we made an appointment and met at the university to start building the site. First, we distributed our responsibilities, who will do what. Each of us individually worked on a given page using all three languages ​​HTML, CSS and Java Script. Later, each participant finished with their page, they sent all the codes to one of the team members - Asim Asgarov. He combined all the codes into one big site. And later we started working on its design. As you can see, we have added icons, gifs, links, animations and the like. Having finished with this, the participant Melek Hajili registered on the educational platform GitHub and posted a post about our project. And the participant Gunel Dibirova posted a post in a twitter, also telling about the site. Having finished with all the work done, we wrote a report on our project and talked about how each of the participants contributed to the construction of the site. And here is the end result of the hard work of all four participants in front of your eyes!
